Detailed Description

Class that provides a setup wizard for the ZeroSync desktop client.

When the user starts ZeroSync for the first time an instance of ZSSetupWizard will start. This setup wizard is used to set the ZeroSync folder and the preferred synchronization options. It has to be finished in order to use ZeroSync. When the wizard gets aborted, ZeroSync quits and the next time the user starts ZeroSync, the setup wizard will start again.
